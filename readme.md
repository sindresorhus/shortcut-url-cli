# shortcut-url-cli

> Get the URL from a [web shortcut](https://en.wikipedia.org/wiki/File_shortcut) file


## Install

```
$ npm install --global shortcut-url-cli
```


## Usage

```
$ shortcut-url --help

  Usage
    $ shortcut-url <file-path>

  Example
    $ shortcut-url google
    https://google.com

  Leave out the extension for cross-platform compatibility
```
